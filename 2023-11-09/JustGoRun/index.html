JustGoRun

Update At:2023年11月9日 中午

今天是2023年11月1日，星期三，最近工作上没啥事，就请假一天。关了闹钟，早上睡到快九点了，真是爽翻了。

一直想买一个run 后缀的域名，之前买了kungfu.run ，到期之后我还是放弃他了。也关注过一些比较好的run 后缀域名，比如 keep run ，just run ，happy run，letsgo run ，但是都可遇不可求，在上次十一回家期间，想到了justgo 这个，就决定拿下他，关键这个域名还没有被注册。当时想下手，但还是犹豫，想等双十一活动，今天看到西部数码有活动，冲动之下，花103块钱注册了3年，本以为占了大便宜，后来想想，也还是没捞到大的好处，本来第一年注册才9块钱，合着后来的两年也是47一年，也没有太便宜。但是木已成舟，just let it go.

可能这个域名只是一个垫脚石或者跳板，未来那些我钟意的还会想办法收入囊中。不过眼下有它就够了。

just go run.对得起这几个单词和一百多块钱。

路漫漫其修远兮，吾将上下而求索。
