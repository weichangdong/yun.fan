龙年归来

Update At:2024年3月2日 中午

今天是2024年2月24号，星期六，正月十五，今天回北京了，春节的假期归零。

昨天早上九点多从县城家里出发的，本来想坐高速的，但是最近下雪，高速都封闭了，退了高速买了火车票，多亏买的早，要不然估计回不来了，因为整个封路导致除了火车没别的渠道去西安了。

等我打车到火车站的时候，看见了火车站人虽然不是非常多，但是是我在这个火车站见到的最多的一次。

关键时刻还是得靠老铁啊！

坐火车的代价就是要在西安多等好几个小时，因为只有一趟从县城到西安的列车。13点就到了西安火车站，我的票是20点的，就附近找了个宾馆，看名字叫 悦东 就去住了，便宜是便宜，环境太差了，下次交钱之前一定得看下房间情况，还好我就临时呆5个小时。放了行李，不想在火车站附近吃饭，虽然有很多家饭馆都营业着，就走了很远，好不容易找着一个面馆，吃了一碗心心念的油泼面，就是有点咸，别的还不错。

吃完就回旅店，房间没空调，有些冷，就和着衣服睡了一觉，睡得倒挺好，一个半小时，再就和家里视频聊天一个小时，很快就到了6点，就出去吃晚饭了。

今天早上九点多就到了北京西站了，倒了两趟公交，顺便吃个饭，回家洗澡洗衣服，睡一觉，一切都要回归平常了。

今年回家，襄阳媳妇家和我老家都去了，在襄阳花了不少钱，走了十来家亲戚，对我来说，走亲戚真的是很不情愿，但是没办法，作为女婿有这个责任，我也只能随大流。老家花钱也有几千吧，过年，对成年人来说，真不是容易过的。

把我买的三星手机给了我妹，去年把我的摩托罗拉手机给了我妈，两年买的两个安卓手机，都是2000左右，结果都没用多久，三星的那个屏幕是真好，不输于苹果，几次升级系统之后，慢成马了，现在又只有一个手机了，我的苹果十二被迫双卡双待营业。

这回家的20多天，彻底放飞自我，好多坚持都落下了，是时候重整河山了。

龙年，希望我的baby 顺利平安诞生，同时多愉悦自己，just go run lets go fun。

我还是个孩子，但是我已经老了。
