关于博客的美

Update At:2023年10月16日 上午

最近在V站看到了好多人在折腾博客，有些样式的确挺好看的，有的是wordpress的，有的是typecho的，多数还是人家自己动手写的。

对于自己动手写的那些，我深表羡慕，从事程序员这么多年，对前端技术一直不熟悉，也很有抵触，也不愿意去了解，而后端的语言，php golang Java Python 我都有涉猎，而且还是比较愿意去学习的，而前端那些乱七八糟的，我总是拒而远之退避三舍。

别人的漂亮炫酷拿不来，就只好安慰自己了，内容至上。一些typecho 等php的博客系统，好看的主题还是不少的，搭建对于我来说也不在话下，就是需要服务器资源，而且要想博客访问速度快，服务器的配置还不能太低，所以我只能望洋兴叹。我现有的博客，虽然做不到花里胡哨，但是我觉得也挺好看的，没必要整天想着向别人看齐，等那一天逼不得已学会了前端，自己在想着重整河山。

博客重要的是坚持写，能长久的存在，不像好多网址今天张牙舞爪极尽奢华明天就404不知所踪了，所以博客能最低成本的活着很重要，就像沙漠中的仙人掌，扛得住风沙耐得住干旱，不会轻易地死去，这才是真正的王者。
